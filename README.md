# Fortnite Drop Discord-bot
A simple discord bot that will help you choose a location to land in your next match in Fortnite by randomly choosing from the available locations.

### Use : 
1. Add the bot to your server from the [link](https://discordapp.com/oauth2/authorize?&client_id=561904430690336778&scope=bot&permissions=0)
2. Use command !drop to get a random drop location.
